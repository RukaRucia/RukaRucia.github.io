# 『Refills～在异世界身为天才魔法使的生存方式～』


&emsp;&emsp;你曾经做过梦。<br>
&emsp;&emsp; 我如今做着梦。<br>
&emsp;&emsp;分不清是谁梦见了什么，也不知是谁先开始做梦的。<br>
&emsp;&emsp;总之，你在梦中遇到我，亦或我在梦中成为你。<br>
<br>
&emsp;&emsp;十七岁的女高中生，多良间蕾娜（Darama Rena），在梦中穿越到异世界，成为天才魔法使雷纳尔多•洛佩兹（Renaldo Lopez）。<br>
<br>
&emsp;&emsp;——那就是，我。<br>
<br>
&emsp;&emsp;请看吧，我的生存方式（Refills）。<br>
&emsp;&emsp;然后总有一天，我一定会让它开启某人的梦。<br>

---

- 【Lucia的一言】

      抱着写个老土的异世界轻小说的念头开始写的东西，很可惜不怎么老土。各方面都很温和的故事，但构成故事的要因说出来估计会非常黑暗。角色都很温柔，大概没关系。
    
      总之，请抱着轻松乐观的心态来观看。如果有什么觉得极度真实或者不真实的东西，请认为它来自夢楽咲流霞的真实生活。
   
    **「——欢迎来到『Refills』的世界！！！」**

## 角色介绍

### <font color = #FF6347 >多良间蕾娜（Darama Rena）</font>

    17岁的日本女高中生。
    各项指标都在中上级别，擅长搏击、绘画和熬夜。歪门邪道很多，羞耻心微弱。在学校朋友很多。喜欢宅文化，经常玩各类galgame和乙女游戏，熟知各种套路，喜欢在现实中进行妄想并对号入座。
    穿越到异世界的魔法天才雷纳尔多·洛佩兹身上，遗憾的是只有体质得到了继承。成为帅哥之后一心想活用脸蛋和社交技巧在异世界大玩特玩，好奇心加倍。
    在学校最好的朋友分别叫做优美（Yumi）和真希（Maki），梦想是和她们一起去东京做漫画家。
    生日是7月2日。


### <font color = #66CDAA >Micaela Garcia Roderick/米卡艾拉·加西亚·罗德里克</font>

    罗德里克亲王家的三女儿。希利德斯凯学院中等部A班特许免课生。
    性格古怪的模范优等生。看上去面无表情，同时经常沉默。个性非常严肃，说任何玩笑话可能都会当真。美貌受到周围人的一致好评。
    特性是粒子光，本人在此之上开发了特殊光剑。

### <font color = #B22222 >Craig/克雷格</font>
    
    贫民窟出身的孤儿，被大魔导师挖角进入希利德斯凯学院。中等部A班的风纪委员。
    不谙世事的老好人，难以拒绝别人的要求。除了魔法之外尤其擅长文书工作。
    特性是切断。

### <font color = #F08080 >Elliot Clark/艾利欧特·科拉可</font>

    雷纳尔多的仆人。希利德斯凯学院中等部A班学生。
    潜行水平极高，胆小怕事，总是默默无闻，看上去很好欺负。
    特性是自然之灵。

### <font color = #7D9EC0 >Zachary Ross/扎卡里·罗斯</font>
    
    希利德斯凯学院中等部A班学生。
    右半边的脸上有类似疤痕的东西，一般会用纱布遮住。擅长辩论，思考事情的方式很复杂。因为独特的思辩方式而被卡尔顿和克拉伦斯称作“军师”。
    无特性特化，但是理解能力强。擅长狙击。

### <font color = #4169E1 >Clarence Foster/克拉伦斯·佛斯特</font>

    希利德斯凯学院中等部A班班长。
    性格开朗而且活跃，一直被认为是乐于助人的善良好青年。
    特性是核变。

### <font color = #8B5A00 >Carlton Finn/卡尔顿·芬恩</font>

     银行家芬恩的长子，有一个妹妹。希利德斯凯学院中等部A班学生。
     钱很多。个性嚣张而聒噪，喜欢没事找事。
     特性是弹性。

### <font color = #339966 >Aisling Evans/阿诗玲·埃文斯</font>

    希利德斯凯学院中等部A班特别培养生。据说老家在国外的偏远小部落。
    放在人群中不太起眼，实际上脸很有少年感。个头不高，身形瘦弱。不会主动和人搭话。
    特性是植物和奏乐。绝对乐感。

### <font color = #ff3366 >Desiree Daphne/黛丝莉·达芙妮</font>

    雷姆利亚国首席执行官家的次女。希利德斯凯学院中等部A班学生。
    开朗热情，经常被好奇心驱使着做事。过于缺少生活常识 闹出笑话是家常便饭。语尾的“嘛↑？”是特点。
    特性是燃烧。

### <font color = #EEDC82 >Jaylen Jones/杰棱·琼斯</font>
  
    希利德斯凯学院中等部A班教会见习生，雷纳尔多的邻座。
    毒舌，没有除了克拉伦斯之外的朋友。比同龄的女生还要麻烦一倍，并且怕生。
    特性是修复和光。

### <font color = #EE6A50 >Elysia Campbell/埃莉西亚·坎普贝尔</font>

    富商坎普贝尔家的次女。希利德斯凯学院中等部A班学生。
    自来熟。温柔天然，眼睛大而无神。经常带来糖果、曲奇或者布丁一类的零食。
    特性是拉力。


### <font color = #9FB6CD >Lottie Gray/罗缇·格雷</font>

     希利德斯凯学院中等部A班学生。家住在花园巷杂货铺。
     头发遮住眼睛，开口时吞吞吐吐，极力避免和人说话。经常被误解。
     特性是感觉和占卜，直觉向来很准。察言观色的能力也很强。

### <font color = #00FFFF >Terrance Wright/特岚斯·莱特</font>
     
    莱特公爵家的长子，有两个妹妹。希利德斯凯学院中等部B班学生。
    骑士团预备成员。有很幼稚的胜负欲，擅自把雷纳尔多视作假想敌，在背后拼命努力。
    特性是熔化，被称作“溢融之莱特”。

### <font color = #B03060 >Cayla/凯拉</font>
     
    从黑街逃进雷姆利亚王都的贫民窟少女，被塞到希利德斯凯学院中等部D班改造。
    经常随意地翻墙进出学校。讨厌肢体接触，有针对他人的洁癖和担心隐私泄露的强迫症。喜欢酸奶和薄荷糖。
    特性是微弱的体能增强和漂浮。大腿内侧有一道很长的伤疤。


### <font color = #FFD700 >Riccardo Vidal Alasdair/里卡尔多·威达尔·阿拉斯代尔</font>

    雷姆利亚国王子。
    时常面带微笑的温柔青年，被认为是下任国王的唯一候选人。衣领上戴着徽章。

### <font color = #EE6A50 >Aylin Richardson/艾琳·理查德森</font>

    希利德斯凯学院中等部A班的班主任。
    巨乳美女。做事由着性子来，时而温和时而暴躁，心情好的时候提过分的要求也会答应。

### <font color = #668B8B >Bruce Miller/布鲁斯·米勒</font>
    
    希利德斯凯学院的校医。
    异常讨厌吵闹，耳边只要有声音就难以集中精神。主业是学者，致力于魔法和生物的研究，在撰写博物志。

### <font color = #D02090 >Celeste Lopez/茜莱丝特·洛佩兹</font>

    雷纳尔多的母亲，担任雷姆利亚国公文审查员。
    在工作上认真负责，受到女王器重。容易感到过度的焦虑和抑郁，常常为不着边际的事情担心，对雷纳尔多有过度保护的倾向。
    没有任何魔法天赋的一般人。

### <font color = #3366cc >Ernesto/爱内斯托</font>

    负责为雷纳尔多做心理诊查的医生。
    亲切而爱好和平，完全相信多良间蕾娜的说辞。

### <font color = #993366 >Miriam Alasdair/米莉亚姆·阿拉斯代尔</font>

    雷姆利亚国女王，旧姓柏克莱（Berkeley），子爵的独生女。16岁和前代国王帕克斯顿·阿拉斯代尔（Paxton Alasdair）订婚，在其死后代理国事，因此成为雷姆利亚国最年轻的女王。
    调整过方针政策之后民众的生活质量得到显著提升，获得压倒性的支持。

### <font color = #ff6600 >Stephanie/斯蒂芬妮</font>
    
    16岁，独自居住在郊外的女巫。

### <font color = #FFAEB9 >Felice/菲莉丝</font>

    雷姆利亚教会所信仰的女神。

### <font color = #5D478B >Renaldo Lopez/雷纳尔多·洛佩兹</font>

    17岁的魔法天才，多良间蕾娜占据的身体的原主。据说性格乖僻。
    生日是11月14日。
